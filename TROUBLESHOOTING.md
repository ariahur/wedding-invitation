# 문제 해결 가이드

## 화면이 보이지 않는 경우

### 1. 개발 서버 실행 확인
```bash
npm start
```
서버가 정상적으로 실행되면 `http://localhost:3000`에서 접근 가능합니다.

### 2. 브라우저 콘솔 확인
브라우저 개발자 도구(F12)를 열고 Console 탭에서 에러 메시지를 확인하세요.

### 3. 네트워크 탭 확인
개발자 도구의 Network 탭에서 CSS나 JS 파일이 제대로 로드되는지 확인하세요.

### 4. 캐시 삭제
브라우저 캐시를 삭제하고 강력 새로고침(Ctrl+Shift+R 또는 Cmd+Shift+R)을 시도하세요.

### 5. 환경변수 확인
`.env` 파일이 있는지, 그리고 Supabase 환경변수가 설정되어 있는지 확인하세요.
(환경변수가 없어도 화면은 보여야 합니다. RSVP 제출만 작동하지 않습니다.)

### 6. 빌드 확인
```bash
npm run build
```
빌드가 성공하는지 확인하세요.

### 7. 포트 확인
다른 프로세스가 3000번 포트를 사용하고 있는지 확인하세요.
```bash
lsof -ti:3000
```
