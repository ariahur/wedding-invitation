### 그라데이션 (Gradients)

#### 로딩 화면 텍스트 그라데이션
```css
linear-gradient(135deg, 
  #1A2F4A 0%, 
  #1A2F4A 5%, 
  #C9A77C 50%, 
  #E6D8C3 100%
)
```
- 방향: 왼쪽 상단 → 오른쪽 하단 (135deg)
- Navy → Gold → Beige 전환
- 로딩 화면 메시지 텍스트에 적용

#### 섹션 구분선 (Section Divider)
```css
linear-gradient(to right,
  rgba(201, 167, 124, 0.6) 0%,
  #C9A77C 10%,
  #E6D8C3 25%,
  #E6D8C3 75%,
  #C9A77C 90%,
  rgba(201, 167, 124, 0.6) 100%
)
```
- 방향: 좌 → 우 (to right)
- 골드/베이지 톤의 섹션 구분선
- About Us, Directions, RSVP, Timeline 섹션 사이에 사용

#### Hero Boarding Pass 액센트 보더
```css
linear-gradient(90deg,
  rgba(201, 167, 124, 0.9) 0%,
  #C9A77C 15%,
  #D4B896 30%,
  #D4B896 70%,
  #C9A77C 85%,
  rgba(201, 167, 124, 0.9) 100%
)
```
- 방향: 좌 → 우 (90deg)
- 골드 톤의 액센트 그라데이션

#### 타임라인 선 (Timeline Line)
```css
linear-gradient(to bottom,
  rgba(224, 224, 224, 0.3) 0%,
  #e0e0e0 20%,
  #e0e0e0 80%,
  rgba(224, 224, 224, 0.3) 100%
)
```
- 방향: 상 → 하 (to bottom)
- 회색 톤의 타임라인 수직선 (섹션 구분선과 일관성 유지)

#### 타임라인 이벤트 라벨
```css
linear-gradient(135deg, 
  #B8956A 0%, 
  #C9A77C 50%, 
  #D4B896 100%
)
```
- 방향: 왼쪽 상단 → 오른쪽 하단 (135deg)
- 카멜 → 골드 → 라이트 골드 전환
- 타임라인 이벤트 배지에 사용

#### Timeline 섹션 배경 (주석 처리됨)
```css
linear-gradient(135deg, #f5f7fa 0%, #c3cfe2 100%)
```
- 방향: 왼쪽 상단 → 오른쪽 하단 (135deg)
- 연한 블루-그레이 그라데이션
- 현재는 주석 처리되어 사용되지 않음

#### Contact Modal 배경
```css
linear-gradient(135deg, #1A2F4A 0%, #2a3f5a 100%)
```
- 방향: 왼쪽 상단 → 오른쪽 하단 (135deg)
- 네이비 블루 그라데이션

## 📐 색상 사용 가이드

### 주요 컬러 스킴
- **Primary**: `#1A2F4A` (Navy Blue) - 주요 액센트, 타이틀, 버튼
- **Secondary**: `#C9A77C` (Gold/Beige) - 키워드, 강조 요소
- **Background**: `#FAF8F3` (Ivory) - 메인 배경
- **Surface**: `#FFFFFF` (White) - 카드 및 섹션 배경
- **Accent**: `#E6D8C3` (Beige) - 섹션 래퍼 배경
- **Divider**: `#e0e0e0` (Light Gray) - 구분선, 타임라인 선, 경계선
- **Camel**: `#B8956A` (Camel) - 타임라인 카운터, 이벤트 라벨

### 텍스트 계층 구조
1. **Primary Text**: `#333` - 가장 중요한 텍스트
2. **Secondary Text**: `#666` - 일반 설명 텍스트
3. **Tertiary Text**: `#999` - 보조 정보, 플레이스홀더
4. **Accent Text**: `#1A2F4A` - 강조 텍스트
5. **Gold Text**: `#C9A77C` - 특별한 강조 (키워드 등)
6. **Camel Text**: `#B8956A` - 타임라인 카운터 전용

### 상태별 색상
- **Success**: `#4CAF50` / `#16a34a`
- **Error**: `#ef4444` / `#dc2626`
- **Hover**: `#2a3f5a` (네이비), `rgba(26, 47, 74, 0.05)` (라이트)
- **Disabled**: `#ccc`

### 섹션별 색상 사용

#### Hero Boarding Pass
- 배경: `#FAF8F3` (Ivory)
- 타이틀: `#1A2F4A` (Navy)
- 액센트 보더: 골드 그라데이션 (`#C9A77C` → `#D4B896`)

#### About Us
- 배경: 흰색 (`#FFFFFF`)
- 구분선: 골드/베이지 그라데이션
- 카드 보더: 회색 (`#e0e0e0`)
- 키워드: 골드 (`#C9A77C`)

#### Directions
- 배경: 흰색 (`#FFFFFF`)
- 구분선: 골드/베이지 그라데이션
- 아이콘: 지하철 `#4A90E2`, 버스 `#F44336`

#### RSVP
- 배경: 흰색 (`#FFFFFF`)
- 구분선: 골드/베이지 그라데이션
- 버튼: 네이비 (`#1A2F4A`)
- 성공/에러: 그린/레드

#### Timeline
- 배경: 흰색 → 종이색 그라데이션 (`#fff 0%` → `#fff 40%` → `#FAF8F3 100%`)
- 구분선: 회색 (`#e0e0e0`)
- 타임라인 선: 회색 (`#e0e0e0`)
- 타임라인 마커: 회색 보더 (`#e0e0e0`)
- 카운터 숫자: 카멜 (`#B8956A`)
- 이벤트 라벨: 카멜/골드 그라데이션

#### Loading Screen
- 배경: `#FAF8F3` (Ivory)
- 텍스트 그라데이션: Navy → Gold → Beige
- 별 파티클: 골드 (`rgba(201, 167, 124, 0.9)`)

## 🎯 색상 조합 예시

### 조화로운 조합
- **네이비 + 골드**: `#1A2F4A` + `#C9A77C` (고급스러운 조합)
- **아이보리 + 베이지**: `#FAF8F3` + `#E6D8C3` (따뜻한 조합)
- **네이비 + 화이트**: `#1A2F4A` + `white` (명확한 대비)
- **회색 + 카멜**: `#e0e0e0` + `#B8956A` (깔끔한 조합)

### 그라데이션 패턴
- **베이지 계열**: `#E6D8C3` → `#F0E6D6` (섹션 배경)
- **골드 계열**: `#C9A77C` → `#D4B896` (액센트 보더)
- **네이비 계열**: `#1A2F4A` → `#2a3f5a` (모달 배경)
- **카멜/골드 계열**: `#B8956A` → `#C9A77C` → `#D4B896` (타임라인 라벨)
- **로딩 텍스트**: `#1A2F4A` → `#C9A77C` → `#E6D8C3` (Navy → Gold → Beige)

## 📝 참고사항

- 모든 색상은 16진수 코드로 정의되어 있습니다.
- 투명도가 필요한 경우 `rgba()` 형식을 사용합니다.
- 그라데이션은 주로 `135deg` (왼쪽 상단 → 오른쪽 하단) 또는 `90deg` (좌 → 우) 방향을 사용합니다.
- 모바일 최적화를 위해 충분한 대비율을 유지합니다.
- 타임라인 섹션은 회색 구분선과 카멜 액센트를 사용하여 깔끔하고 따뜻한 느낌을 줍니다.
- 로딩 화면은 Navy → Gold → Beige 그라데이션으로 우아한 전환 효과를 제공합니다.
